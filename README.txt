Business Cards by Demitry Gavrisenko.

This is a business cards project, created in React, using Material-UI.

It includes the following:

Main layout with header and navbar menu, page with content, footer with links to pages.
Page navigation is done with React routes. It is possible to login/sign-up, create new cards, edit existing cards, like cards, delete cards. 

Light/Dark mode included. Search is implemented. It's possible to view user info, logout. 

Within the business cards file are included: client with source code and server with back end code and basic database - template users and cards.

If admin is logged in, additional page is displayed - sandbox with various pages that demonstrate functionalities and various techniques in React.

Node modules are not included in the client and server folders.
